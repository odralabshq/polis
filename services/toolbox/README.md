# Toolbox Service

Administrative and interactive tools for the Polis environment.

## Features

- **MCP Agent**: Secure bridge for AI agents to interact with the Polis system.
- **Approve CLI**: Command-line tool for managing HITL approvals.
- High-level orchestration for async tasks and administrative intake.

## Components

- `crates/mcp-agent/`: Rust implementation of the Polis MCP protocol.
- `crates/approve-cli/`: CLI tool for security administrators.
